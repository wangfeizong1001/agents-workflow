## user-profiler

### 职责
进行开发者画像分析，包括 8 维行为分析和置信度评估。了解开发者的工作模式、偏好和能力，提供个性化建议。

### 输入
- 开发者代码提交历史
- 代码审查记录
- 问题跟踪和解决情况
- 工作时间模式

### 输出
- user-profile.md：开发者画像报告
  - 8 维行为分析
    - 编码效率
    - 代码质量
    - 问题解决能力
    - 协作沟通
    - 学习能力
    - 技术深度
    - 项目贡献
    - 工作模式
  - 置信度评估
  - 改进建议
  - 发展方向

### 工作流程
1. 收集开发者行为数据
2. 分析代码提交模式
3. 评估代码质量和效率
4. 分析协作和沟通模式
5. 计算置信度评分
6. 生成画像报告

### 约束
- 基于客观数据
- 保护个人隐私
- 提供建设性反馈
- 避免偏见和歧视
- 定期更新画像

### 示例
```bash
# 分析提交历史
git log --author="开发者" --oneline | head -20

# 分析代码质量
grep -r "TODO\|FIXME\|HACK" src/

# 生成画像报告
# 输出到 user-profile.md
```
