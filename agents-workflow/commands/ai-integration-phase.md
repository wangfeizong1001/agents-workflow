## /ai-integration-phase 命令

### 用法
/ai-integration-phase [AI 功能]

### 参数
- `--model`: AI 模型选择
- `--api`: API 集成方式
- `--prompt`: 提示词工程

### 行为
1. 分析 AI 需求
2. 选择合适的模型
3. 设计集成架构
4. 实现 API 调用
5. 测试和优化

### 示例
```
/ai-integration-phase chatbot --model gpt-4 --api rest
```

### 输出
AI 集成代码和配置文件
