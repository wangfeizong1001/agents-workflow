## /discovery-phase 命令

### 用法
/discovery-phase [主题]

### 参数
- `--depth`: 探索深度（quick/deep/exhaustive）
- `--sources`: 信息来源
- `--output`: 输出格式

### 行为
1. 收集相关信息
2. 分析现有解决方案
3. 识别技术约束
4. 评估可行性
5. 生成发现报告

### 示例
```
/discovery-phase ai-integration --depth deep
```

### 输出
发现报告，包含研究结果、风险评估和建议
