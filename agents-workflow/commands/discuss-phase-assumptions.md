## /discuss-phase-assumptions 命令

### 用法
/discuss-phase-assumptions [阶段名称]

### 参数
- `--list`: 列出所有假设
- `--validate`: 验证假设
- `--update`: 更新假设

### 行为
1. 识别阶段假设
2. 分析假设有效性
3. 讨论假设影响
4. 验证关键假设
5. 更新假设文档

### 示例
```
/discuss-phase-assumptions feature-design --validate
```

### 输出
假设分析报告和验证结果
