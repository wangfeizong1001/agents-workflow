## /discuss-phase 命令

### 用法
/discuss-phase [议题]

### 参数
- `--stakeholders`: 参与者
- `--format`: 讨论格式（brainstorm/debate/review）
- `--timebox`: 时间限制

### 行为
1. 准备讨论材料
2. 引导结构化讨论
3. 记录关键观点
4. 达成共识
5. 生成讨论纪要

### 示例
```
/discuss-phase api-design --format brainstorm --timebox 30min
```

### 输出
讨论纪要，包含决策、行动项和后续步骤
