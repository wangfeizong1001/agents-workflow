## /explore 命令

### 用法
/explore [探索主题]

### 参数
- `--depth`: 探索深度
- `--scope`: 探索范围
- `--output`: 输出格式

### 行为
1. 定义探索目标
2. 收集相关信息
3. 分析探索结果
4. 生成探索报告
5. 提供后续建议

### 示例
```
/explore "微服务架构" --depth deep --scope technical
```

### 输出
探索报告，包含发现和建议
