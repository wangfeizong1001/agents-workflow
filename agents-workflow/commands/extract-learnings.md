## /extract-learnings 命令

### 用法
/extract-learnings [来源]

### 参数
- `--source`: 来源类型
- `--categorize`: 分类
- `--export`: 导出格式

### 行为
1. 分析来源内容
2. 识别经验教训
3. 分类整理
4. 提取最佳实践
5. 生成经验文档

### 示例
```
/extract-learnings session --source commits --categorize
```

### 输出
经验教训文档和最佳实践
