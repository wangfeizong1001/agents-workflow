## /fast 命令

### 用法
/fast [快速操作]

### 参数
- `--mode`: 快速模式
- `--skip`: 跳过步骤
- `--optimize`: 优化选项

### 行为
1. 识别快速操作
2. 优化执行路径
3. 跳过非必要步骤
4. 快速执行操作
5. 验证结果

### 示例
```
/fast deploy --mode minimal --skip tests
```

### 输出
快速操作结果
