## /help 命令

### 用法
/help [命令名称]

### 参数
- `--list`: 列出所有命令
- `--category`: 按分类筛选
- `--search`: 搜索命令

### 行为
1. 显示命令帮助信息
2. 列出可用命令
3. 提供使用示例
4. 解释参数选项
5. 提供相关命令

### 示例
```
/help --list --category 计划类
/help code-review
```

### 输出
命令帮助信息和使用指南
