## /inbox 命令

### 用法
/inbox [操作]

### 参数
- `--list`: 列出所有项
- `--add`: 添加新项
- `--resolve`: 解决项目
- `--archive`: 归档项目

### 行为
1. 查看收件箱内容
2. 分类处理项目
3. 分配优先级
4. 跟踪处理进度
5. 归档已处理项目

### 示例
```
/inbox --list
/inbox --add "修复登录问题"
```

### 输出
收件箱内容列表和处理状态
