## /mvp-phase 命令

### 用法
/mvp-phase [产品名称]

### 参数
- `--focus`: MVP 重点功能
- `--scope`: 范围限制
- `--timeline`: 时间限制

### 行为
1. 识别核心价值主张
2. 筛选最小功能集
3. 定义 MVP 范围
4. 制定快速迭代计划
5. 设定成功指标

### 示例
```
/mvp-phase saas-app --focus user-onboarding --timeline 2weeks
```

### 输出
MVP 计划文档，包含核心功能列表和迭代计划
