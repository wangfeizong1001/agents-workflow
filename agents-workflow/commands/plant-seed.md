## /plant-seed 命令

### 用法
/plant-seed [想法/概念]

### 参数
- `--context`: 背景信息
- `--potential`: 潜力评估
- `--next-steps`: 下一步

### 行为
1. 记录初始想法
2. 分析潜在价值
3. 识别必要条件
4. 规划培育路径
5. 创建种子文档

### 示例
```
/plant-seed "AI 驱动的代码审查工具" --context 团队需求
```

### 输出
种子文档，包含想法描述、潜力评估和培育计划
