## /profile-user 命令

### 用法
/profile-user [用户标识]

### 参数
- `--analyze`: 分析维度
- `--segment`: 用户分群
- `--export`: 导出格式

### 行为
1. 收集用户数据
2. 分析用户行为
3. 生成用户画像
4. 识别用户特征
5. 提供个性化建议

### 示例
```
/profile-user user-123 --analyze behavior,preference --export json
```

### 输出
用户画像报告和分析结果
