## /progress 命令

### 用法
/progress [项目/任务]

### 参数
- `--detail`: 详细程度
- `--compare`: 比较基准
- `--forecast`: 预测完成时间

### 行为
1. 收集进度数据
2. 分析完成情况
3. 识别阻塞问题
4. 预测完成时间
5. 生成进度报告

### 示例
```
/progress feature-auth --detail high --forecast
```

### 输出
进度报告，包含完成百分比和预测
