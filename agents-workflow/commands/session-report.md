## /session-report 命令

### 用法
/session-report [会话标识]

### 参数
- `--include`: 包含内容
- `--format`: 报告格式
- `--share`: 分享选项

### 行为
1. 收集会话数据
2. 分析工作成果
3. 总结关键决策
4. 记录经验教训
5. 生成会话报告

### 示例
```
/session-report today --include commits,tasks --format markdown
```

### 输出
会话报告，包含工作成果和经验总结
