## /sketch-wrap-up 命令

### 用法
/sketch-wrap-up [草图会话]

### 参数
- `--decisions`: 记录决策
- `--next`: 下一步
- `--archive`: 归档草图

### 行为
1. 总结草图成果
2. 记录关键决策
3. 识别后续行动
4. 归档草图材料
5. 生成总结报告

### 示例
```
/sketch-wrap-up dashboard-design --decisions --next
```

### 输出
草图总结报告和后续计划
