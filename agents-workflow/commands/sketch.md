## /sketch 命令

### 用法
/sketch [概念]

### 参数
- `--style`: 草图风格
- `--variants`: 变体数量
- `--interactive`: 交互式

### 行为
1. 理解概念需求
2. 生成设计变体
3. 创建草图原型
4. 收集反馈
5. 迭代优化

### 示例
```
/sketch "用户仪表板" --variants 3 --interactive
```

### 输出
设计草图和反馈报告
