## /spike-wrap-up 命令

### 用法
/spike-wrap-up [尖峰会话]

### 参数
- `--findings`: 发现总结
- `--recommendations`: 建议
- `--evidence`: 证据

### 行为
1. 总结尖峰成果
2. 记录技术发现
3. 评估可行性
4. 提供决策建议
5. 生成总结报告

### 示例
```
/spike-wrap-up tech-evaluation --findings --recommendations
```

### 输出
尖峰总结报告和决策建议
