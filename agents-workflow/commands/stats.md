## /stats 命令

### 用法
/stats [统计范围]

### 参数
- `--period`: 时间周期
- `--metrics`: 指标类型
- `--format`: 输出格式

### 行为
1. 收集统计数据
2. 计算关键指标
3. 分析趋势变化
4. 生成统计报告
5. 提供可视化

### 示例
```
/stats --period 7days --metrics commits,tasks --format chart
```

### 输出
统计报告和可视化图表
