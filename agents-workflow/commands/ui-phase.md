## /ui-phase 命令

### 用法
/ui-phase [页面/组件名称]

### 参数
- `--design`: 设计文件路径
- `--framework`: UI 框架
- `--responsive`: 响应式设计

### 行为
1. 分析设计需求
2. 创建 UI 组件结构
3. 实现交互逻辑
4. 适配不同屏幕尺寸
5. 生成 UI 代码

### 示例
```
/ui-phase login-page --framework react --responsive
```

### 输出
UI 组件代码和样式文件
