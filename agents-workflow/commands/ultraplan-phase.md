## /ultraplan-phase 命令

### 用法
/ultraplan-phase [项目名称]

### 参数
- `--horizon`: 计划时间范围
- `--granularity`: 计划粒度
- `--risk`: 风险评估

### 行为
1. 全面分析项目目标
2. 制定多层次计划
3. 建立里程碑体系
4. 识别关键路径
5. 生成超级计划文档

### 示例
```
/ultraplan-phase enterprise-app --horizon 6months --granularity weekly
```

### 输出
超级计划文档，包含战略、战术和操作计划
