## chinese-documentation

### 用途
中文文档排版参考——中英文空格、全半角标点、术语保留、链接格式、中文文案排版指北约定。

### 使用方式
1. 应用排版规则
2. 检查标点使用
3. 验证术语格式
4. 优化链接格式
5. 生成排版报告

### 输入
- 中文文档
- 排版规范

### 输出
- 排版后的文档
- 排版报告
- 修复建议

### 示例
```markdown
## 排版规则

### 中英文空格
- ✅ 使用 React 框架
- ❌ 使用React框架

### 标点符号
- ✅ 这是一个句子。
- ❌ 这是一个句子.

### 术语保留
- ✅ 支持 TypeScript
- ❌ 支持类型脚本

### 链接格式
- ✅ [文档](https://example.com)
- ❌ 文档(https://example.com)
```
