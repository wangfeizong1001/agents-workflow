## design-consultation

### 用途
设计咨询：理解你的产品、研究领域全貌、提出完整的设计系统（美学、字体、颜色、布局、间距、动效），并生成字体+颜色预览页面。创建 DESIGN.md 作为项目设计的唯一真相源。

### 使用方式
1. 理解产品定位和目标用户
2. 研究行业设计趋势
3. 定义设计系统
4. 创建 DESIGN.md
5. 生成预览页面

### 输入
- 产品描述
- 目标用户画像
- 品牌调性
- 可选：现有设计参考

### 输出
- DESIGN.md（设计系统文档）
- 颜色调色板
- 字体规范
- 间距系统
- 预览 HTML 页面

### 示例
```markdown
# DESIGN.md - 我的产品

## 美学
- 风格: 现代极简
- 调性: 专业、可信、友好

## 颜色
- Primary: #2563EB (蓝色)
- Secondary: #10B981 (绿色)
- Neutral: #F3F4F6

## 字体
- 标题: Inter (700)
- 正文: Inter (400)

## 间距
- 基础单位: 4px
- 组件间距: 16px
```
