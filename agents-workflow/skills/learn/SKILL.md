## learn

### 用途
管理项目 learnings。回顾、搜索、清理和导出在各次会话中积累的知识。在用户询问过去的模式或念叨 "didn't we fix this before?" 时主动建议。

### 使用方式
1. 回顾历史会话
2. 提取关键学习
3. 分类和标签
4. 搜索和检索
5. 清理过时知识

### 输入
- 搜索查询
- 可选：时间范围
- 可选：分类过滤

### 输出
- 学习清单
- 搜索结果
- 知识图谱

### 示例
```markdown
## 项目学习

### 最近学习
1. [性能] 使用 Redis 缓存减少数据库查询
2. [安全] JWT 需要实现 token 黑名单
3. [架构] 微服务间使用 gRPC 通信

### 搜索结果
查询: "数据库优化"
- 2024-01-10: 添加了复合索引
- 2024-01-05: 实现了查询缓存
```
