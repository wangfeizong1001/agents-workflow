## Git 助手

> **触发条件**：需要 Git 操作辅助时使用
> **预计耗时**：10分钟
> **产出物**：Git 操作结果

### 阶段 1：检查（3分钟）

- ② git-status → 检查仓库状态
- 门禁：状态已检查

### 阶段 2：操作（5分钟）

- ② git-add → 添加文件
- ② git-commit → 提交更改
- 门禁：操作完成

### 阶段 3：验证（2分钟）

- ② git-log → 验证提交
- 门禁：验证通过

### 回滚方案
- 使用 git-reset 撤销
