## 导入

> **触发条件**：需要导入外部资源时使用
> **预计耗时**：15分钟
> **产出物**：导入的资源

### 阶段 1：收集（5分钟）

- ② import → 收集导入源
- ② explore → 分析资源内容
- 门禁：资源已确认

### 阶段 2：处理（6分钟）

- ② apply-change → 应用导入
- 门禁：导入已处理

### 阶段 3：整合（4分钟）

- ② verify-change → 验证整合
- ② map-codebase → 更新映射
- 门禁：整合完成

### 回滚方案
- 删除导入的资源
