## 移除工作区

> **触发条件**：需要删除工作区时使用
> **预计耗时**：8分钟
> **产出物**：工作区已安全移除

### 阶段 1：检查（3分钟）

- ① list-workspaces → 确认要移除的工作区
- ② git-status → 检查未提交更改
- 门禁：确认可以安全移除

### 阶段 2：备份（3分钟）

- ① context-save → 保存当前上下文
- ② git-stash → 暂存未完成工作
- 门禁：重要数据已备份

### 阶段 3：移除（2分钟）

- ① remove-workspace → 执行移除操作
- ② cleanup → 清理相关文件
- 门禁：工作区已完全移除

### 回滚方案
- 从 git 历史恢复工作区
