## UI 阶段

> **触发条件**：需要设计和实现用户界面时使用
> **预计耗时**：90分钟
> **产出物**：完整的 UI 实现

### 阶段 1：设计（25分钟）

- ① design-consultation → 设计咨询
- ② design-shotgun → 生成设计变体
- 门禁：设计方案已确定

### 阶段 2：实现（50分钟）

- ① design-html → 实现 UI 代码
- ② apply-change → 应用设计变更
- 门禁：UI 已实现

### 阶段 3：审查（15分钟）

- ① design-review → 设计审查
- ② plan-design-review → 设计评审
- 门禁：UI 通过审查

### 回滚方案
- 恢复到设计前状态
